# 🌌 The Multiverse Chatbot

A stateful, interactive AI chatbot built with Streamlit and the Gemini API. 

Users can toggle between different AI personalities (like Ironman or a Hacker) and adjust their emotional intensity using a slider. The app utilizes `st.session_state` to maintain an active "Memory Vault," ensuring the bot remembers the context of the conversation even as parameters change.
